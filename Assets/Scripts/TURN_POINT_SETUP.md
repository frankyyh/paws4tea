# 狗狗转折点设置指南

现在狗狗可以在到达指定位置后改变方向继续移动。

## 📋 工作原理

1. 狗狗从右侧生成，向左移动
2. 当狗狗到达设定的转折点（Turn Point X）时
3. 狗狗会改变方向，按照设定的新方向继续移动
4. 速度保持不变

## ⚙️ 在Unity中设置

### 步骤 1: 设置DogSpawner

1. 在 **Hierarchy** 中选中 **DogSpawner** GameObject
2. 在 **Inspector** 中找到 **Dog Spawner** 组件
3. 找到 **Turn Point** 部分：

   - **Turn Point X**: 转折点的X坐标
     - 例如：`0`（屏幕中间）
     - 例如：`-2`（屏幕中间偏左）
     - 例如：`2`（屏幕中间偏右）
  
   - **Turn Direction**: 转折后的移动方向
     - 点击右侧的箭头展开
     - **X**: 水平方向（-1向左，0无水平移动，1向右）
     - **Y**: 垂直方向（-1向下，0无垂直移动，1向上）
     - **Z**: 保持为 0

### 步骤 2: 常用方向设置

#### 向下移动（推荐）
- **Turn Direction**:
  - X: `0`
  - Y: `-1`
  - Z: `0`

#### 向上移动
- **Turn Direction**:
  - X: `0`
  - Y: `1`
  - Z: `0`

#### 继续向左（不改变方向）
- **Turn Direction**:
  - X: `-1`
  - Y: `0`
  - Z: `0`

#### 向右移动
- **Turn Direction**:
  - X: `1`
  - Y: `0`
  - Z: `0`

#### 向左下移动（对角线）
- **Turn Direction**:
  - X: `-0.707`（或 -1）
  - Y: `-0.707`（或 -1）
  - Z: `0`

## 📝 示例设置

### 示例 1: 到达屏幕中间后向下移动

**DogSpawner设置：**
- **Turn Point X**: `0`
- **Turn Direction**: 
  - X: `0`
  - Y: `-1`
  - Z: `0`

**效果：** 狗狗从右侧出现，向左移动到屏幕中间（X=0），然后向下移动。

### 示例 2: 到达X=-2后向上移动

**DogSpawner设置：**
- **Turn Point X**: `-2`
- **Turn Direction**: 
  - X: `0`
  - Y: `1`
  - Z: `0`

**效果：** 狗狗从右侧出现，向左移动到X=-2的位置，然后向上移动。

## ⚠️ 注意事项

- **Turn Point X** 应该小于生成位置（Spawn X），因为狗狗从右向左移动
- 如果 **Turn Point X** 设置得太大（大于生成位置），狗狗永远不会到达转折点
- **Turn Direction** 的X和Y值可以是任意数字，但通常使用 -1, 0, 1 或它们的组合
- 速度在转折前后保持不变
- 如果不想使用转折功能，可以将 **Turn Direction** 设置为 (0, 0, 0)，或者将 **Turn Point X** 设置为一个很大的负数

## 🎮 测试建议

1. 先设置一个简单的转折点（例如：X=0，向下移动）
2. 运行游戏，观察狗狗是否在正确位置转折
3. 根据需要调整 **Turn Point X** 和 **Turn Direction**
4. 可以在Scene视图中看到转折点的位置（X坐标）
